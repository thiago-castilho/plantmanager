## 🚀 Getting started

Clone the project and access the folder

```bash
$ git clone https://github.com/thiago-castilho/plantmanager.git && cd plantmanager
```

Follow the steps below

```bash
# Install the dependencies
$ yarn install

# Start the project
$ expo start
```

## 💻 Project

App to always remember to water your plants 🌱
